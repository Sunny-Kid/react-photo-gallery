# react-photo-gallery

![][license-url]

A photo gallery application based on React

## Install

**yarn**

```bash
yarn install
```

**npm**

```bash
npm install
```

## Develop

```bash
yarn dev
```

## Build

**Development**

```bash
yarn build:dev
```

**Production**

```bash
yarn build:prod
```
